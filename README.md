# Logistics & Freight Forwarding B2B Landing Page

Compact, high-performance one-page web application for a freight forwarding and logistics company operating across Ukraine and Europe. Built with a mobile-first approach, focusing on conversion, speed, and accessibility.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Integrations:** Resend SDK (Lead capture form via Email)
- **Deployment:** Vercel
